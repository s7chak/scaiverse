The Writing Pipeline

Some text here for testing purposes of the writ pipeline to publish workflow.

A pipeline for having written in markdown.

Process

  • It gets translated to rich HTML with images in some cases for scaiverse (portfolio website blogs)
  • Also a final markdown copy gets forever backed up in Obsidian's 4Blogged folder with the link of the scaiverse blog URL
  • The goal is to streamline publishing and make it easy with one click or push of a button from Insyt.

All orchestrated from one place.
